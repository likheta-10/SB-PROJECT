# Sentence-BERT Concept Evaluation

## Project Overview
This project evaluates a student's explanation by comparing it with a reference concept using Sentence-BERT embeddings and cosine similarity.

## Features
- Sentence-BERT embedding generation
- Cosine similarity calculation
- Score normalization (0–100)
- Simple Flask web interface

## Technologies Used
- Python
- Flask
- Sentence-Transformers
- Scikit-learn
- NumPy

## Installation

```bash
pip install -r requirements.txt
python app.py
```
